# A simple and clean Todo Application built with React.

This project allows users to:

➕ Add new tasks

✅ Mark tasks as completed

❌ Delete tasks

📋 Manage daily activities in an organized way

## Tech Stack

JavaScript (ES6+)

React

Vite (for fast development and build)

## Purpose

The goal of this project was to practice:

React components

State management (useState)

Event handling

Working with lists and keys

This is a beginner-friendly project focused on strengthening core React fundamentals.
